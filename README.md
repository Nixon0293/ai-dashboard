# AI Dashboard — дашборд заказов

## Описание
Дашборд для визуализации заказов из RetailCRM. Данные хранятся в Supabase, график построен с помощью Chart.js, деплой на Vercel. Telegram-бот присылает уведомления о новых заказах.

## Технологии
- RetailCRM API
- Supabase (PostgreSQL)
- Python (requests)
- HTML, JavaScript, Chart.js
- Vercel (деплой)
- Telegram Bot API

## Как создавался проект
Проект выполнялся в рамках тестового задания. Для генерации кода, написания скриптов и отладки использовался DeepSeek (AI-ассистент). Основные этапы:
1. Регистрация аккаунтов (RetailCRM, Supabase, Vercel, Telegram Bot)
2. Загрузка тестовых заказов в RetailCRM через API
3. Перенос заказов из RetailCRM в Supabase
4. Создание HTML-дашборда с графиком и деплой на Vercel
5. Настройка Telegram-бота для уведомлений

## Промпты для AI
- «Напиши Python-скрипт для загрузки заказов из JSON в RetailCRM через API»
- «Сделай дашборд на HTML с графиком заказов из Supabase»
- «Настрой Telegram-бота для уведомлений о заказах»

## Проблемы и решения
1. **Ошибка авторизации в RetailCRM** — помогла документация API, использовал параметр `apiKey`
2. **Двойное объявление переменных в HTML** — переписал файл с нуля, убрал дубли
3. **Ошибка с supabaseJs vs supabase** — исправил вызов на `supabase.createClient()`
4. **GitHub не принимает пароль** — создал персональный токен доступа
5. **Ошибка с limit в RetailCRM** — изменил параметр с 10 на 20, 50 или 100
